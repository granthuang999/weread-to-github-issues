# 将微信读书状态自动同步到issues并自动渲染成网页

## 项目说明

- 从https://github.com/sailor0913/weread-to-notion获得启发并且fork源仓库自行修改修改。修改后本仓库已经不具备同步到Notion功能，请务必注意。
- 通过 github actions 每天定时同步
- 同步微信读书部分元数据信息
